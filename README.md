### About the program
#### Program "Brain games" consists of five mathematical mini-games.
### Requirements:
#### PHP 7
#### Composer 2.5.8
### Install:
#### 1. Clone your own repository
#### 2. Go to working directory
#### 3. Use make install
#### 4. Launch any of the five games
* make brain-even
* make brain-calc
* make brain-gcd
* make brain progression
* make brain-prime
